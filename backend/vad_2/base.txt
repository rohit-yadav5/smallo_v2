                  └───────────────────────┬──────────────────────────┘
                                          │
                        IF vad_prob < threshold → IGNORE (noise)
                                          │
                                          ▼
               ┌────────────────────────────────────────────────────────┐
               │     3. Speech-Type Classifier (Speech vs Noise)        │
               │  - CNN / Wav2Vec2 checks if it is REAL HUMAN SPEECH   │
               │  - Output: speech_type = {speech/noise/background}    │
               │  - Confidence threshold = 0.7                          │
               └──────────────────────┬─────────────────────────────────┘
                                      │
                     IF NOT real speech → IGNORE (sneeze, cough, etc)
                                      │
                                      ▼
         ┌──────────────────────────────────────────────────────────────────┐
         │                 4. Relevance Detection Module                    │
         │                                                                  │
         │   4A. Wake-word Detector                                         │
         │       - “Hey Bot”, “Listen”, etc                                 │
         │       ────────────────────────────────────────────────┐         │
         │                                                       ▼         │
         │   4B. Direction of Arrival (Optional)                 Accept if  │
         │       - Checks if sound comes from user direction     any of     │
         │       ────────────────────────────────────────────────┤         │
         │                                                       │         │
         │   4C. Tiny ASR + Intent Classifier                    │         │
         │       - Checks if speech is relevant to bot           ▼         │
         └──────────────────────────┬──────────────────────────────────────┘
                                    │
                     IF NOT relevant → IGNORE and bot continues talking
                                    │
                                    ▼
            ┌─────────────────────────────────────────────────────────┐
            │                 5. Decision Engine                      │
            │  Combines: VAD + Speech-Type + Relevance + Smoothing   │
            │                                                        │

            │  ELSE → CONTINUE BOT SPEAKING                         │
